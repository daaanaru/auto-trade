テクニカルBUYが出た銘柄のみ、企業の財務データをチェック。スコア > 0 でエントリー許可。

  • PER < 15 → +0.3（割安）
  • PBR < 1.5 → +0.2（資産価値に対して安い）
  • 配当利回り > 3% → +0.2
  • 売上成長 > 10% → +0.3
  • 52週高値の95%超 → -0.1（高値圏で危険）
  • 時価総額 < $1B → -0.1（小型でリスク大）

ペーパートレード実行 (1時間後に自動)

screenerの1時間後にunified_paper_trade.pyが同じシグナルを再確認し、ポジションを取る。

  • 1銘柄 ≤ ポートフォリオの20%
  • 同時保有 ≤ 5ポジション
  • 損切り: -5% / 利確: +10%で半分
  • 3/30に卒業判定 → 合格なら実弾移行

用語解説

RSI（相対力指数）

直近の値上がり幅と値下がり幅の比率。0〜100の範囲。30以下は「売られすぎ」で反発しやすい。70以上は「買われすぎ」で下がりやすい。

ボリンジャーバンド（BB）

移動平均線の上下に標準偏差（σ）の幅を足した帯。価格がバンドの外に出ると「行きすぎ」で戻りやすい。

MFI（マネーフローインデックス）

出来高を加味したRSI。お金の流入量で売られすぎ/買われすぎを判断する。

ダイバージェンス

価格の動きと指標の動きが逆方向になること。例: 価格が安値更新したのにMFIは上がっている → 「そろそろ反転するかも」のサイン。

EMA（指数移動平均）

直近の価格を重視した移動平均。200日EMAの上にいれば上昇トレンド、下にいれば下降トレンド。

PER / PBR

PER = 株価 ÷ 1株あたり利益。低いほど割安。PBR = 株価 ÷ 1株あたり純資産。1以下なら資産価値以下で買える。
